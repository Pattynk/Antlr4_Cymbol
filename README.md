Para probar este programa, se debe de cargar todo en una carpeta
Abrir terminal desde la carpeta en donde se tenga el archivo o en visual studio code darle click a nueva terminal
Si se abre en visual studio code, deberá de instalarse una extensión antes:
https://marketplace.visualstudio.com/items?itemName=mike-lischke.vscode-antlr4

____________________________

Agregar de uno por uno los siguientes comandos:

antlr4 Cymbol.g4
compile *.java
grun Cymbol file -gui t.cymbol


____________________________

Si lo que se desea es probar los demás ejercicios, entonces cerrar el árbol generado y volver a colocar las instrucciones pero usando 

grun Cymbol file -gui t2.cymbol

o

grun Cymbol file -gui t3.cymbol

según sea el archivo
